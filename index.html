Purrfect Pals Adoption

This cat adoption tracking website uses APIs from animal shelters to show you pals in nearest to your zip code! It takes data from each cat to give you a price range estimate. You can also set up an account through Google to save pets, take notes, and send the cutest cats to a leaderboard gallery!

  • This project was made with Abilena Thomas and Jessie Whiting, and our repo is here.
  • We used C# for our internal API and getting OAuth tokens
  • Angular as our web framework
  • SQL our accounts, shelter data, and cutest cats list. Hosted on Azure.
